kubelife host - The Ultimate Web App Hosting Tool

Version: 1.0.0

Author: Devesh Singh Rathore

Introduction

Welcome to kubelife host, the all-in-one solution for hosting your web applications effortlessly. Whether you're a developer looking to quickly spin up a static website or an administrator preparing for dynamic deployments, kubelife host has you covered.

This tool leverages the power of Docker and Docker Compose to deliver fast, reliable, and seamless hosting solutions. With an intuitive interface and robust automation, kubelife host transforms your hosting experience into a streamlined, hassle-free process.

Why Choose kubelife host?

  • Ease of Use: Simplified commands for both beginners and experts.
  • Flexibility: Host static websites today, dynamic applications tomorrow.
  • Automation: Automatically generates and manages Docker configurations.
  • Customization: Configure ports, image names, and more to suit your needs.
  • Error Handling: Detailed prompts and validations ensure no deployment mistakes.
  • Future-Ready: Scalable architecture to support dynamic web applications.

Key Features

1. Static Website Hosting

Host lightweight static websites with:

  • HTML-only apps
  • CSS-based designs
  • Combinations of HTML and CSS

The tool automates:

  • Creation of Dockerfile and docker-compose.yml files.
  • Validation of directories and critical files like index.html.
  • Deployment using Docker Compose with automatic build and start.
2. Dynamic Web Application Hosting (Coming Soon)

Prepare to deploy complex dynamic applications in future updates.

3. Port Management

Fully customizable container and host ports. Built-in validation ensures correct configurations.

4. Kubernetes Naming Compliance

Ensures all process names meet Kubernetes naming conventions for compatibility with cloud-native platforms.

5. Robust Validation
  • Prerequisite checks for Docker and Docker Compose.
  • File and directory validation to avoid deployment issues.

System Requirements

Before using kubelife host, ensure the following are installed:

  • Docker: Install Docker
  • Docker Compose: Install Docker Compose

How It Works

Step 1: Run the Command

kubelife host

Step 2: Choose the Application Type

  • 1 for Static Website Hosting.
  • 2 for Dynamic Web Application Hosting (Coming Soon).

Step 3: Configure Deployment

  • Enter your process name, ensuring it follows Kubernetes conventions.
  • Provide the directory path containing your web application files.
  • Choose the host and container ports for the application.

Step 4: Automatic Deployment

The tool generates all necessary configurations (Dockerfile, docker-compose.yml) and deploys your application using Docker Compose.

Step 5: Access Your Application

http://localhost:

  • Prerequisite Checks: If Docker or Docker Compose is missing, the tool exits with a detailed error message.
  • Process Name Validation: Ensures process names are Kubernetes-compliant.
  • Directory and File Validation: Checks for the presence of index.html in the specified directory.

FAQs

Q1: What if Docker is not installed?
Docker is not installed. Please install Docker and try again.
Q2: Can I use this tool on Windows?

Yes, as long as Docker and Docker Compose are installed and configured properly.

Q3: What happens if I provide an invalid process name?
Invalid process name. It must contain lowercase alphanumeric characters, hyphens, and periods.
Q4: How can I change the port number?

Ports can be customized during the configuration phase of the deployment.
