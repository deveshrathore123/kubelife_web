KubEdge Documentation

Introduction to KubEdge

Welcome to the official KubEdge documentation. KubEdge is an innovative cloud computing platform built with Kubernetes and Docker to automate cloud deployments, scalability, and management. It streamlines workflows and integrates monitoring tools like Prometheus and Grafana.

Installation and Setup

To get started with KubEdge, follow these steps:

  1. Install Docker on your local system.
  2. Install Kubernetes using a tool like Minikube.
  3. Clone the KubEdge repository:
    4. git clone https://github.com/deveshrathore123/kubedge_server.git
  4. Run the deployment scripts to set up your KubEdge environment.

How to Use KubEdge

KubEdge helps automate your cloud workflows by:

  • Using Docker for containerized applications.
  • Utilizing Kubernetes for orchestrating and scaling microservices.
  • Integrating with Prometheus and Grafana for monitoring and analytics.
