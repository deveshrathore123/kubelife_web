KubeLife Documentation

Key Features:

  • Docker & Docker Compose Validation: Ensures Docker and Docker Compose are installed before proceeding.
  • Static Website Hosting: Deploys HTML/CSS websites with Nginx using Docker.
  • Dockerfile & Docker Compose Setup: Automatically generates Dockerfile and docker-compose.yml for hosting the website.
  • Port Configuration: User-defined ports for container access and external access.
  • Future Dynamic Hosting Support: Placeholder for dynamic web app deployment (under development).

Installation & Setup:

Before running the script, ensure that you have the following installed:

  • Docker: To create and run containers.
  • Docker Compose: To manage multi-container Docker applications.

How to Run:

  1. Clone or Download the Script.
  2. Make the Script Executable: chmod +x kubelife.sh
  3. Run the Script: ./kubelife.sh

Static Website Deployment (HTML/CSS)

Choose "1" for static websites when prompted.

Provide the web app information: type (HTML, CSS), directory path, image name, etc.

Example Walkthrough:

./kubelife.sh
Select the application type:
1. Static Website
2. Dynamic Web Application
Enter your choice: 1

Troubleshooting:

  • Docker Not Installed: Instructions will be provided.
  • Invalid Ports: You will be prompted to enter valid ports.
  • Missing index.html: The script will exit with an error.

Future Improvements:

  • Dynamic Web App Deployment
  • SSL/TLS Support
